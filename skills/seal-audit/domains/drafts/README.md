# Draft Domain Checklists

These checklists have not been validated against a real SEAL engagement. They may contain wrong assumptions about what data matters, what questions to ask, or how discovery actually works in the domain.

**Do not use in production SEAL runs without review.**

To graduate a checklist: run a real SEAL Scan on a client in that domain, compare what the checklist asked for vs. what you actually needed, then revise and move to the parent `domains/` folder.
